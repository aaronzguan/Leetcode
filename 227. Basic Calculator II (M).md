## 227

The expression string contains only **non-negative** integers, `+`, `-`, `*`, `/` operators and empty spaces ``. The integer division should truncate toward zero.

```
Input: "3+2*2"
Output: 7

Input: " 3+5 / 2 "
Output: 5
```



```python
class Solution(object):
    def calculate(self, s):
        """
        :type s: str
        :rtype: int
        """      
        stack = []
        num = 0
        # Make the first sign as + because the first num doesnot have sign.
        sign = '+'
        
        for i in range(len(s)):
          if s[i].isdigit():
            num = num*10 + int(s[i])
            
           # the reason for having i = len(s) - 1 here is that we do the last calculation at the end, because usually the last one is a number.
          if not s[i].isdigit() and s[i] != ' ' or i == len(s) - 1:
            if sign == '+':
              stack.append(num)
            elif sign == '-':
              stack.append(-num):
            elif sign == '*':
              stack.append(stack.pop() * num)
            elif sign == '/':
              tmp = stack.pop()
              if tmp // num < 0 and tmp % num != 0:
                	stack.append((tmp//num) + 1)
              else:
                	stack.append(tmp//num)
            # Store the current sign
            sign = s[i]
            num = 0
       return sum(stack)
```
